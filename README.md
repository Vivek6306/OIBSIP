# Web Development Level 2 - Task 1: Calculator

## Objective
Build a fully functional browser-based calculator capable of performing basic arithmetic operations with a user-friendly button interface.

## Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)

## Features

- Display screen that shows the current input and result
- Numeric buttons (0–9) and decimal point
- Addition (+)
- Subtraction (-)
- Multiplication (×)
- Division (÷)
- Equals (=) button
- Clear (C) button
- Backspace/Delete button
- Division by zero handling
- Operator chaining
- CSS Grid layout
- Event listeners (No inline onclick)
- Built without using `eval()`

## Project Structure

```
WebDev-L2-Calculator/
│── index.html
│── style.css
│── script.js
│── README.md
```

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Double-click `index.html` or open it in any web browser.

## Learning Outcomes

- DOM Manipulation
- Event Handling
- CSS Grid
- JavaScript Functions
- Arithmetic Logic
- Error Handling
- Calculator State Management
